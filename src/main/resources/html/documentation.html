Input Data (Record Logs)

A record log combines event data with time-series data in the same log.    

Our program supports record logs contained in a CSV file.

At present the following attributes of a record log are supported: case-id, timestamp, type, activity, and any number of measurement variables. Other attributes will be ignored. Optional Parameters (Start)

By default the parameters for our CSV parser are set to "yyyy-MM-dd HH:mm:ssXXX" for the timestamp and ";" for the CSV delimiter. The reference for the possible timestamp format can be viewed here:

Date and Time Patterns

These parameters can be changed by selecting the corresponding fields before selecting a CSV file.

Additionaly, we implemented a tooltip for the column selection fields when the mouse pointer is located and holded on the corresponding field.

Visualization (Overview)

The event data in a record log is visualized on the dotted chart on the first half of the window and the time-series data on the line chart on the second half.

Our charts support zooming using the mouse wheel and panning by holding down the middle button of the mouse. Additional zoom options are located on the toolbar of each of the charts.

Further our charts offer filter options for the charts. The case-ids of the shown event activities and measurements can be set from the drop-down menu on the top of the window. The activities on the dotted chart can also be filtered and the measurement variables can be added to the line chart .
